# crop-scan

This suite of fine-tuned Generative AI models enhances accuracy in detecting diseases and classifying plants/crops using leaf images.

## Data source

The dataset used as the source for this project is from Kaggle: https://www.kaggle.com/datasets/salmasyed1360/plant-diseases-100k-labelled-images?resource=download
